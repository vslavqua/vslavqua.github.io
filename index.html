Infrastructure Automation with Docker and Ansible

This documentation describes how to fully automate a test infrastructure with five Linux containers using Docker and Ansible. The setup includes local NTP servers per datacenter and Zabbix Agent2 for monitoring, while accessing external repositories through a proxy server.

Introduction

Introduction

The purpose of this project is to automate the deployment and configuration of a Linux-based infrastructure that simulates two datacenters (AAA and ZZZ). Each datacenter includes local NTP servers and monitored hosts using Zabbix Agent2.
